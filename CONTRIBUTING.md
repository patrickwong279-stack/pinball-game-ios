# Contribution Guidelines

Thank you for considering contributing to our project! Here are some guidelines to help you get started.

## Reporting Bugs
- Please search the [issues](https://github.com/patrickwong279-stack/pinball-game-ios/issues) to see if your bug has already been reported.
- If it hasn't, open a new issue and include a detailed description of the bug, steps to reproduce, and any screenshots if applicable.

## Suggesting Features
- We welcome feature requests! Please open an issue and provide a detailed description of your suggested feature, including why it would be beneficial and how it could be implemented.

## Submitting Code
- Fork the repository and create a new branch for your feature or bugfix (e.g., `feature/my-feature` or `bugfix/fix-bug`).
- Make your changes, and ensure that your code adheres to our coding standards.
- Write tests for your changes if applicable. We strive for good test coverage.
- Submit a pull request and include a description of the changes you've made.

## Code of Conduct
We are committed to creating a positive and inclusive environment for all contributors. Please adhere to our Code of Conduct:
- Be respectful and considerate to others.
- Be mindful of your language and actions.
- We do not tolerate harassment or discrimination in any form.

Thank you for your contributions!
